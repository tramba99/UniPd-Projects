Istruzioni per applicare l’algoritmo VAD ad un file inputaudioN.data:
1) Inserire il file inputaudioN.data all’interno della stessa cartella di vad.m
2) Aprire il file vad.m con Matlab
3) Premere su “Run”
4) Inserire, come richiesto, N (dove N è l’indice dell’input, da 1 a 5) per selezionare il file di input inputaudioN.data a cui si vuole applicare l'algoritmo
5) Osservare il file di output prodotto: outputVADN.txt.
